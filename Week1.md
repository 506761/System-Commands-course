* ### uname-a (linux kernel version)
* ### pwd (present working directory)
* ### ls 
  - a : all . displays hidden files 
  - l : use a long listing format
  - i : print index number of each file (inode)
* ### rm 
  - remove a file 
* ### mv
  - move , rename
